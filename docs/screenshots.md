# UI Prototype Screenshots

## Design Overview

AcademiaConnect UI prototypes were created using Figma, following Material Design principles and ensuring responsive layouts.

## Key Screens

### 1. Login / Sign Up
- Clean, minimal design
- SSO button prominent
- Local account option available

### 2. Student Dashboard
- Timeline of notices (center)
- Quick access to chat (sidebar)
- Course filters (top bar)
- Upcoming deadlines widget

### 3. Chat Interface
- WhatsApp-inspired familiar layout
- Friend list on left
- Chat messages on right
- Attachment support

### 4. Notice Board
- Grid or list view toggle
- Filter panel
- Search bar
- Category badges

### 5. Notes Repository
- Search-first interface
- Filter by course/tags
- Preview cards
- Download/view buttons

### 6. User Profile
- Profile picture
- Academic information
- Privacy settings
- Activity history
