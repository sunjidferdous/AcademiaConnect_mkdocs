# Friend Management

## FR-003: Friend Request System

### Description
Enable users to send, receive, accept, and reject friend requests, managing their academic social network.

### Key Features
- Send friend requests
- Accept/reject incoming requests
- View friend list
- Unfriend capability
- Search users
- Profile visibility controls

### Request Workflow
1. User A sends request to User B
2. User B receives notification
3. User B accepts or rejects
4. If accepted, both users added to each other's friend lists
5. Chat becomes available between friends

### Acceptance Criteria
- Friend requests sent successfully
- Notifications for new requests
- Accept/reject works correctly
- Friend list displays all accepted friends
- Search finds users by name/email
